# The Ninja Way - Naruto mod for Terraria

## Установка:
1. Скачай `TheNinjaWay.tmod`
2. Положи в `Documents/My Games/Terraria/tModLoader/Mods`
3. Включи в игре

## Управление:
- Z - Каменная стена
- X - Огненный шар
- V - Ветряная пуля
- C - Водяные пули
- F - Молния
- K - Шаринган
- J - Бьёкуган
- L - Врата Ли / способность клана
- M - Мангекё
- N - Сусаноо
- B - Камуи
- Y/U/I - Риннеган
- T/G - Каге (30 уровень)
- H - Показать чакру
